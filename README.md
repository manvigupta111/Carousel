# React Image Carousel

A simple and responsive Image Carousel built with React.  
This project demonstrates image sliding functionality using React Hooks like `useState`, `useEffect`, and `useRef`.

---

## Features

- Automatic image slideshow
- Next and Previous navigation buttons
- Infinite looping carousel
- Pause slideshow on hover
- Resume slideshow on mouse leave
- Responsive full-width image display
- Clean and minimal UI

---

## Technologies Used

- React
- JavaScript (ES6+)
- CSS
- React Hooks (`useState`, `useEffect`, `useRef`)
